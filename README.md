

# ChatSy App

**ChatSy** is a simple, functional chat application built in Java for Android devices. It enables users to send and receive messages in real-time, providing a seamless communication experience. The app is designed with a clean interface and is suitable for learning and small-scale chat projects.

---

## Features

- Real-time messaging
- User-friendly interface
- Basic chat functionalities
- Easy to extend or customize

---

## Technologies Used

- Java
- Android SDK
- XML for layouts

---

## Getting Started

### Prerequisites

- Android Studio (latest version recommended)
- Java Development Kit (JDK)
- An Android device or emulator for testing

### Installation

1. Clone the repository:
```bash
git clone https://github.com/afshansif/Chatsy.git
```

2. Open the project in Android Studio:
   - Launch Android Studio
   - Open the cloned folder as a project

3. Build and run:
   - Connect your Android device or start an emulator
   - Click on the "Run" button in Android Studio
